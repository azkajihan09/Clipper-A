# SmartClip Setup Instructions

## Required Dependencies

This application requires the following dependencies that are not included in the repository due to their large size:

### 1. FFmpeg
- Download from: https://www.gyan.dev/ffmpeg/builds/
- Extract to: `BACA DAHULU PENTING !/ffmpeg/`
- Required files:
  - `bin/ffmpeg.exe`
  - `bin/ffprobe.exe`

### 2. Python 3.11
- Download from: https://www.python.org/downloads/
- Install Python 3.11.0 or later
- Make sure to add Python to PATH during installation

### 3. Python Dependencies
After installing Python, run:
```bash
pip install -r requirements.txt
```

## Quick Setup Script
Run `INSTALL_SMARTCLIP.bat` as administrator to automatically download and install dependencies.

## Usage
After setup is complete, run `START_SMARTCLIP.bat` to start the application.
